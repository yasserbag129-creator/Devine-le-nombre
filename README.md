# Devine le Nombre - Python

Petit projet Python pour apprendre :
- Les fonctions
- Les boucles (`while`)
- Les conditions (`if/elif/else`)
- La gestion des entrées utilisateur
- La génération de nombres aléatoires avec `random`

## Comment jouer
1. Exécute `python devine_nombre.py`
2. Suis les instructions à l'écran
3. Devine le nombre, le jeu te donnera des indices
4. Essaie de trouver en le moins d'essais possible !
